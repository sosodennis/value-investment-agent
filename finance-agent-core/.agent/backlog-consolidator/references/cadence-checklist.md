# Cadence Checklist

Use this every consolidation cycle.

## Pre-Cycle
1. Confirm scope boundary for this cycle.
2. Confirm owner and update window.
3. Confirm output mode (artifact only vs direct file edits).

## During Consolidation
1. Build complete source inventory.
2. Normalize and deduplicate items.
3. Validate superseded candidates with evidence checklist.
4. Produce master artifact with fixed status sections.

## Post-Cycle
1. Record archived docs (if any).
2. Record unresolved conflicts/open assumptions.
3. Set next review date.
